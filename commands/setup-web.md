# setup-web — Web Addon Setup

Reads `addons/web/addon.yml` from the plugin and applies all Web-specific configuration to the current project.

## Status

This command is a placeholder for future use. The Web addon (`addons/web/addon.yml`) is not yet available.

## What This Command Will Do

1. Read addon manifest from `${CLAUDE_PLUGIN_ROOT}/addons/web/addon.yml`
2. Register any Web-specific MCP servers
3. Deploy Web-specific scripts
4. Configure hooks if needed
5. Display setup summary
